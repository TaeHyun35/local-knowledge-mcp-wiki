## Notes

OWASP LLM Top 10 describes common risk categories for LLM applications, including prompt injection, sensitive information disclosure, supply chain issues, excessive agency, and model behavior risks. This source is used for LLM-specific risk pages.

## Candidate Concepts

- Prompt injection
- Sensitive information disclosure
- Excessive agency
- Tool permission boundary
- Model output validation
